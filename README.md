**About**

Bakeotopia is a modern bakery web application built with a modern frontend stack. It showcases a bakery-themed UI/UX and can serve as a portfolio or starter project for a bakery storefront.

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone https://github.com/dhwani1608/bakeotopia-a_bakery_app.git

# Step 2: Go into the project directory.
cd bakeotopia-a_bakery_app

# Step 3: Install dependencies.
npm i

# Step 4: Start the development server.
npm run dev

#Step 5: Open in brower
Usually at http://localhost:3000
```
## What technologies are used for this project?

- React
